OTHER HTML TAGS REFERENCE

The <h1> to <h6> tags

Tag Name Description
<h1> Heading 1 A first level heading.
<h2> Heading 2 A second level heading
<h3> Heading 3 A third level heading.
<h4> Heading 4 A fourth level heading
<h5> Heading 5 A fifth level heading.
<h6> Heading 6 A sixth level heading

The <p> Tag

Tag Name Description
<p> Paragragh This is a tag for displaying a paragraph.

Text Formatting tags

Tag Name Description
<i> Italic This is used to make text italic.
<b> Bold This is used to make text bold.
<em> Emphasis This is to make emphasis with a text.
<strong> Strong This is used to highlight important text.
